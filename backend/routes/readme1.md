# Restaurant MERN App

## 📌 Description

A full-stack restaurant web application built using MERN stack (MongoDB, Express, React, Node.js).

## 🚀 Features

* User authentication
* Food menu display
* Order system
* Admin panel (if added)

## 🛠️ Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB

## ▶️ How to Run

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm start

## 📂 Project Structure

* /frontend → React app
* /backend → Node server
